## TPA_BOT
TPA 알고리즘 스터디 주간 코드리뷰를 위한 알림봇입니다.

### [구현 기능] 
**1) 시간 알림**
- 코드리뷰 당일 및 코드리뷰 시작 알림
- 무작위 리뷰 순서 지정

**2) 동작 확인**  
- /ruthere 커맨드로 봇의 동작 여부를 확인합니다.  

**3) 리뷰 일정 변경**  
- /schedule-meeting 커맨드로 코드 리뷰 일정을 변경합니다.  
- /current-meeting 커맨드로 현재 코드 리뷰 일정을 확인합니다.
